🎬 Power BI Project: Movie Sales Analysis

This project presents an interactive Movie Sales Analysis Dashboard built using Power BI, focusing on extracting meaningful insights from movie data through data visualization and storytelling.

📊 Project Overview

The dashboard analyzes key aspects of movies such as revenue, ratings, genres, votes, and yearly trends. It helps in understanding patterns in the film industry and identifying top-performing movies, genres, and directors.

📁 Dataset Features

The dataset includes the following attributes:

Rank
Title
Genre
Description
Director
Actors
Year
Runtime (Minutes)
Rating (IMDb)
Votes
Revenue (Millions USD)
Metascore

📈 Key Analysis Performed

1. Movie Overview & Time Trends Number of movies released each year Year-wise revenue trends Average IMDb rating over time Average runtime analysis Top years with highest revenue and movie releases
2. Genre & Rating Analysis Distribution of movies across genres Highest-rated genres Revenue contribution by genre Average metascore by genre Genres with highest audience votes
3. Top Performers & Box Office Insights Directors generating highest revenue Most voted movies Top 10 highest-grossing movies Revenue vs rating comparison
4. Correlation & Advanced Metrics Relationship between revenue, rating, votes, and runtime Average revenue and rating trends Revenue distribution by genre and year
Overall KPIs like average revenue, metascore, and runtime

📊 Dashboard Features
Interactive filters (slicers)
Bar charts, column charts, and line charts
Funnel chart and donut chart
KPI cards for quick insights
Clean and user-friendly layout

🚀 Tools Used
Power BI
Data Cleaning & Transformation (Power Query)
Data Visualization & DAX

💡 Key Insights
Identified top-performing genres and directors
Observed trends in movie releases and revenue growth
Found relationships between ratings, votes, and revenue
Highlighted high-performing movies based on multiple metrics
